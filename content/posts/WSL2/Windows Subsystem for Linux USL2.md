## Windows Subsystem for Linux USL2  
How-To Guide of Installation.(Ubuntu)

**WSL2 **was announced at Microsoft Build 2019. WSL2 features a Linux kernel running inside Windows 10 and is built on the core technology of Hyper-V to provide better Linux application support and improved file performance. Transitioning to WSL2 is seamless. 
 
This is a Step by Step guide on how to install Ubuntu on your Windows 10 Machine. 

1. Turn On **Windows Subsystem for Linux** on Windows OS 
![1.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775273767/XiTjkZ-ww.jpeg)
![2.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775283596/RSYObTG5v.jpeg)

2.  After WSL 2.0 gets Enabled. It will require a Reboot of Windows OS. 
![3.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775296102/89YDtl-zE.jpeg)
4. Linux Distro is not installed. Tested by running Bash. 
![4.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775311835/rt9AD8qSn.jpeg)  

5. Open Microsoft Store > Download and Install Ubuntu. 
![5.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775322998/DkBC95EI_.jpeg) 
![6.JPG](https://cdn.hashnode.com/res/hashnode/image/upload/v1629775343639/zTpm1S6Dd.jpeg)

6. After Install > Launch it. It will take few minutes to unzip all files locally.
