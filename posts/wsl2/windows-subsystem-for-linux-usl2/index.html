Windows Subsystem for Linux USL2

How-To Guide of Installation.(Ubuntu)

**WSL2 **was announced at Microsoft Build 2019. WSL2 features a Linux kernel running inside Windows 10 and is built on the core technology of Hyper-V to provide better Linux application support and improved file performance. Transitioning to WSL2 is seamless.

This is a Step by Step guide on how to install Ubuntu on your Windows 10 Machine.

  1. Turn On Windows Subsystem for Linux on Windows OS

  2. After WSL 2.0 gets Enabled. It will require a Reboot of Windows OS.

  3. Linux Distro is not installed. Tested by running Bash.

  4. Open Microsoft Store > Download and Install Ubuntu.

The following links will open the Microsoft store page for each distribution:

Ubuntu 18.04 LTS , 20.04 LTS
openSUSE Leap 15.1
SUSE Linux Enterprise Server 12 SP5
Kali Linux
Debian GNU/Linux
Fedora Remix for WSL
Pengwin Enterprise
Alpine WSL

  1. After Install > Launch it. It will take few minutes to unzip all files locally.

  2. Asked for a new Username and Password. And Ubuntu Successfully got installed.

  3. Checkout ls-al command to see the filesystem it has

  4. You can mount local Disks (C:, D:, G:\ Drives) onto this Linux Distro.

  5. Some other commands to test and learn Ubuntu which is locally deployed within your Windows OS.
